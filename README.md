# SalesForce-Status-Cases-Colors v 1.0.1
json & css  Chrome extension to colorize case statuses in Sales Force Lightning


SALES Force THEME
This extension customizes the appearance of selected elements in the Salesforce interface by adding colors to case statuses, making it easier to quickly identify the state of a case.

Key Features:
Works exclusively on Salesforce domains (*.lightning.force.com,  *.force.com).
Does not collect any data or track user activity.
Requires no additional permissions beyond injecting CSS styles.

Why use it?
If you work in Salesforce and want to visually distinguish case statuses faster, this extension is for you.
Security:
The extension does not communicate with external servers, does not store data, and performs no operations other than styling elements on the page.

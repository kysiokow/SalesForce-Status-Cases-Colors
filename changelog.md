Salesforce Theme
SFTheme: Highlights case statuses in Sales Force Lightning with colors.
It is based on CSS code and does not interfere with internal system settings or download/upload any data.


## 1.0.1
- Assigning colors to individual cases in Salesforce:
In Progress, Reopened, Pendingng Response, Request Accepted, Response Received etc.

- Highlights the line with the email attachment
